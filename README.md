# Pong

A simple Pong clone written in C++ using raylib.

## Features
- State-based game loop
- Two-player local controls
- Pause and round reset system
- Collision handling with paddles and walls

## Controls
- W/S: Left paddle
- ↑/↓: Right paddle
- Enter: Start / Resume
- P: Pause

## Build
Requires raylib.
